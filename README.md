## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd shivang_portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite
- `npm eject` - Ejects from Create React App (one-way operation)

## AI Chatbot Features

The portfolio includes an intelligent chatbot that can answer questions about:
- Personal information and contact details
- Technical skills and technologies
- Work experience and current role
- Projects (both professional and personal)
- Education and achievements
- Availability for opportunities

Simply click the AI button in the bottom-right corner to start chatting!

## Deployment

To deploy this portfolio:

1. Build the production version:
```bash
npm run build
```

2. Deploy the `build` folder to your hosting service of choice:
   - GitHub Pages
   - Netlify
   - Vercel
   - AWS S3
   - etc.

## Customization

To customize this portfolio for your own use:

1. Update personal information in all component files
2. Modify the color scheme in CSS files (currently using blue (#4a9eff) and purple (#9333ea))
3. Update the chatbot knowledge base in `ChatBot.js`
4. Replace project details in `Projects.js`
5. Update skills in `Skills.js`
